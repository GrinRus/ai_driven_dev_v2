# AGENTS.md

This directory contains the canonical roadmap and the short actionable backlog.

## Rules

- `roadmap.md` is the source of truth.
- `backlog.md` is only a queue of local task ids.
- Do not add a backlog item unless it already exists in the roadmap.
- Use `.agents/skills/backlog-ops/` and `.agents/skills/task-slicing/` before changing planning structure.
